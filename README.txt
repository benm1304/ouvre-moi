COMMENT LANCER

1. Dézippe le dossier.
2. Ouvre index.html avec Chrome, Edge ou Firefox.

MUSIQUE
Pour ajouter la musique :
1. Mets ton fichier audio dans assets.
2. Renomme-le : music.mp3
3. Rouvre index.html.
4. Clique sur Commencer ou Musique.

NOTE
Il n’y a plus de choix final. Elle lit la lettre, puis le site se termine doucement.
